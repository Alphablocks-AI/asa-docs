# Asa Docs

Merchant help center for [Asa](https://asa.so), built with [Mintlify](https://mintlify.com).

## Local preview

```bash
npm i -g mint
mint dev
```

Open `http://localhost:3000`. Navigation and pages are defined in `docs.json`.

## Structure

- `get-started/` — onboarding guides
- `integrations/` — app connection guides
- `resources/` — pricing, best practices, knowledge source limits
- `legal/` — policies and terms
- `images/` — screenshots referenced in MDX
