Shoppers trust answers backed by real reviews. Connect Yotpo once and Asa can use your review data alongside your Shopify catalog — about three minutes end to end.

<Steps titleSize="h4">

<Step icon="store" title="Open Yotpo in Shopify">

In your **Shopify admin**, go to **Settings > Apps and sales channels**.

Under **Installed apps**, find **Yotpo: Product & Photo Reviews**, open it, then click **Open app** in the top right.

<Frame>
  <img src="/images/yotpo-shopify-apps.png" alt="Shopify Settings showing Yotpo in installed apps" />
</Frame>

</Step>

<Step icon="key" title="Copy your API credentials">

In the **Yotpo** sidebar, go to **Settings > Store Settings > API Credentials**.

Copy both values:

| Field | Where to find it |
|---|---|
| **App Key** | API Credentials page |
| **Secret Key** | API Credentials page |

Example credentials URL:

`https://admin.shopify.com/store/[your-shopify-subdomain]/apps/yotpo/reviews/settings/api-credentials`

<Frame>
  <img src="/images/yotpo-api-credentials.png" alt="Yotpo API Credentials page showing App Key and Secret Key" />
</Frame>

</Step>

<Step icon="plug" title="Connect Yotpo in Asa">

Navigate to **Add Content** in the sidebar, open the **Apps** tab, and click **Connect** on the **Yotpo** card.

<Frame>
  <img src="/images/yotpo-connect-dialog.png" alt="Asa Connect to Yotpo dialog with Yotpo App Key and Yotpo Secret fields" />
</Frame>

Enter the values from Yotpo:

| Field | What to paste |
|---|---|
| **Yotpo App Key** | Your App Key from Yotpo |
| **Yotpo Secret** | Your Secret Key from Yotpo |

Click **Connect**. When setup succeeds, the card shows **Connected**.

<Note>
Only one reviews app can be connected at a time. If you already use Klaviyo Reviews or Judge.me, disconnect that integration before connecting Yotpo.
</Note>

</Step>

</Steps>

## Notes

- Copy **App Key** and **Secret Key** exactly as shown in Yotpo — extra spaces break the connection.
- API credentials give Asa read access to review data only — not full store admin access.
