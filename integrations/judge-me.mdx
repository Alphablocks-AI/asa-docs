Shoppers trust answers backed by real reviews. Connect Judge.me once and Asa can use your review data alongside your Shopify catalog — about three minutes end to end.

<Steps titleSize="h4">

<Step icon="store" title="Open Judge.me in Shopify">

In your **Shopify admin**, go to **Settings > Apps and sales channels**.

Under **Installed apps**, find **Judge.me Reviews**, open it, then click **Open app** in the top right.

<Frame>
  <img src="/images/judgeme-shopify-apps.png" alt="Shopify Settings showing Judge.me Reviews in installed apps" />
</Frame>

</Step>

<Step icon="key" title="Copy your API token and subdomain">

In the **Judge.me** sidebar, go to **General Settings > Integrations**.

Click **View API Token** and copy:

| Field | Where to find it |
|---|---|
| **Private API Access Token** | Shown in the API token dialog |
| **Shopify Subdomain** | The part before `.myshopify.com` in your store URL |

Example integrations URL:

`https://admin.shopify.com/store/[your-shopify-subdomain]/apps/judgeme/shop/general_settings/integrations`

<Frame>
  <img src="/images/judgeme-api-token.png" alt="Judge.me Integrations tab showing Private API Access Token and subdomain" />
</Frame>

</Step>

<Step icon="plug" title="Connect Judge.me in Asa">

Navigate to **Add Content** in the sidebar, open the **Apps** tab, and click **Connect** on the **Judge.me** card.

<Frame>
  <img src="/images/judgeme-connect-dialog.png" alt="Asa Connect to Judge.Me dialog with Shopify Subdomain and Private API Access Token fields" />
</Frame>

Enter the values from Judge.me:

| Field | What to paste |
|---|---|
| **Shopify Subdomain** | Your store subdomain (e.g. `acme` from `acme.myshopify.com`) |
| **Private API Access Token** | The token from Judge.me |

Click **Connect**. When setup succeeds, the card shows **Connected**.

<Note>
Only one reviews app can be connected at a time. If you already use Klaviyo Reviews or Yotpo, disconnect that integration before connecting Judge.me.
</Note>

</Step>

</Steps>

## Notes

- Your **Shopify Subdomain** must match the subdomain in your Shopify admin URL exactly.
- The API token gives Asa read access to review data only — not full store admin access.
