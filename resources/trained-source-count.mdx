Every Asa plan caps how many **knowledge sources** Asa can have trained at once. Each trained source is one row in **Manage Content** — except snippets, which are unlimited and do not count.

<Info>
Plan limits: **50** (Free), **250** (Starter), **500** (Growth). See [Plans and pricing](/resources/plan-pricing) for the full comparison.
</Info>

## What counts as one knowledge source

| Source type | How it is counted |
|---|---|
| **Files** (PDF, DOCX, CSV, TXT) | One file = one knowledge source when trained |
| **Websites** (URL) | One URL = one knowledge source when trained |
| **Shopify — Products** | One product = one knowledge source |
| **Shopify — Blogs** | One blog article = one knowledge source |
| **Shopify — Pages** | One Online Store page = one knowledge source |
| **Shopify — Policies** | One store policy = one knowledge source |
| **Notion** | One selected page = one knowledge source |
| **Google Drive** | One doc or file = one knowledge source |

When you select content to train, the dashboard shows how many **Knowledge source(s)** you are about to use — for example in the Shopify and Notion selection dialogs.

## What does not count separately

These are included with the parent source, not billed as extra knowledge sources:

| Item | Included with |
|---|---|
| **Product metafields** | The Shopify product you trained |
| **Judge.me reviews** | The linked Shopify product |
| **Yotpo reviews** | The linked Shopify product |
| **Klaviyo Reviews** | The linked Shopify product |
| **Snippets** | Do not count toward your plan limit at all |

<Note>
Snippets are added under **Add Content > Snippet**. Use them for short FAQs or macros without using plan capacity. You can publish trained snippets as widget FAQs from **Manage Content**.
</Note>

## Shopify stays in sync without extra sources

When Shopify is connected, product and content updates sync via webhooks. Retraining updates the **same** knowledge source row — it does not create a duplicate count for the same product, page, or article.

<Tip>
Prefer **Add Content > Apps** (Shopify, Notion, Google Drive) over manual uploads when content lives there already. Integrations keep answers current with less manual re-uploading.
</Tip>

## Check your usage

Open **Billing** from your account menu (or subscription settings) and find **Usage Summary**. **Total Knowledge Sources** shows trained count versus your plan limit (for example `42 / 250`).

If training is blocked, the dashboard explains how many sources you have left or prompts you to upgrade.
